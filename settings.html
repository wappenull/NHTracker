Manual hint

Google Takeout browsing history

Inspect your own entire browsing history and populate "READ" book id by searching for
URL pattern like 'nhentai.net/g/NNNNNNN/'
Select BrowserHistory.json and submit it here
It is processed using JavaScript locally in the extension, it is not upload to remote server anywhere, you can check the source code. This is a matter of trust, you and me :)

Using URL file per line

Submit a text file where each line containing URL, the line could include other stuff but extension will search only the URL pattern in each line.
You could export this from history tool or something else. History Trends Unlimited is good tool to export such data from your history.
